# NLog.LoggingScope

Available via NPM
```
install-package NLog.LoggingScope
```

#### Enrich your log entries with identifiers that match the scopes that the entries originate from

```C#
using(new LoggingScope("MyScope"))
{
  Logger.Debug("Hello");
  Logger.Debug("World!");
}

using(new LoggingScope("AnotherScope"))
{
  Logger.Info("Fizz");
  Logger.Info("Buzz");
}
```
